## Global Superstore | Sales & Profit Performance Dashboard

### Tools
- Power BI
- Data Modeling
- DAX (basic measures)

### Features
- KPI cards (Sales, Profit, Margin, Delivery Days, Orders)
- Category & Sub-category analysis
- Country-wise sales map
- Dark mode dashboard design

### Objective
To provide business stakeholders with a clear overview of sales performance, profitability, and geographic distribution.
